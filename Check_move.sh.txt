#!/bin/bash
echo "Enter name of file"
read name
echo "Enter path to a source folder"
read source
echo "Enter path to a destination folder"
read destin

echo "Checking if file exist in destination folder"
if [ -f $destin/$name ]; then
   echo "File $name exists in $destin"
else
   echo "File $name does not exists in $destin"
fi  
   echo "Checking if file exist in source folder"
if [ -f $source/$name ]; then
   echo "File $name exists in $source"
else
   echo "File $name does not exists in $source"
fi
if [ -s $destin/$name ]; then
   echo "Size of file $name is not equal 0"
else
   echo "Size of file $name is equal 0"
fi
